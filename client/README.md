# react-firebase-social-media-app

## 1: API Base URL

Add https://asia-south1-react-social-ape-70daa.cloudfunctions.net/api/ as the 'proxy' value in package.json

## 2: Install packages

run `npm install`

## 3: Run project

run `npm start`

## 4: Open it

go to [http://localhost:3000](http://localhost:3000)
